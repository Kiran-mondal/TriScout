# TriScout Repository Description

**TriScout** is a **fast, modular cybersecurity framework for network reconnaissance** that combines the strengths of multiple programming languages to deliver a comprehensive security scanning solution.

## What it does:
- **Port Scanning:** Leverages Go's concurrency to rapidly scan network ports with goroutines
- **Data Enrichment:** Uses Python for intelligent vulnerability detection and analysis
- **Real-Time Monitoring:** Provides a Node.js-powered web dashboard with live scanning results
- **Multi-Format Reporting:** Exports findings as JSON, CSV, or PDF documents
- **Secure Access:** Implements JWT-based authentication for dashboard protection

## Technology Stack:
The project efficiently uses **five languages**:
- **JavaScript (36.4%)** – Frontend and Express web server
- **Python (21.6%)** – Data processing and vulnerability detection
- **HTML (20.4%)** – Dashboard UI templates
- **Go (12.1%)** – High-performance port scanning engine
- **Shell (9.5%)** – Orchestration and deployment scripts

## Key Features:
- Concurrent network scanning with automatic data pipeline orchestration
- Single-command execution via `run_all.sh` for complete network reconnaissance
- Professional reporting capabilities for security assessments
- Educational and authorized penetration testing tool

The project demonstrates excellent polyglot software architecture, where each language is used for its specific strengths: Go for performance, Python for data science, and Node.js for web UI.
